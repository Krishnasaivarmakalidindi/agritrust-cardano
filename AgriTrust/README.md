# AgriTrust

## 1. Your Project
AgriTrust

## 2. Your Project's Description
AgriTrust is a decentralized agricultural commerce platform where farmers can list produce directly, buyers can place offers, AI supports pricing insights, and Cardano-based escrow helps secure payments and build trust across the transaction flow.

## 3. What Problem Are You Trying to Solve
Traditional agricultural trade often suffers from delayed payments, low transparency, middlemen dependency, and weak trust between buyers and farmers. AgriTrust addresses these issues by creating a verified digital marketplace with escrow-backed transactions, traceable activity, and proof of trade integrity.

## 4. Tech Stack Used While Building the Project
- Frontend: React, TypeScript, Vite, Tailwind CSS
- Backend: Supabase, PostgreSQL, Supabase Auth, Supabase Realtime, Storage
- Blockchain: Cardano, Aiken, MeshJS
- Utilities: React Router, Framer Motion, Lucide React, Zod, QR code generation

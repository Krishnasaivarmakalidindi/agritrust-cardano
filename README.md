# AgriTrust

## Problem Statement

Farmers lose profits due to middlemen, delayed payments, and lack of trust.

## Solution

AgriTrust is a decentralized agricultural marketplace that connects farmers directly with buyers using AI-assisted pricing and Cardano smart contract escrow.

## Features

- Farmer & Buyer Authentication
- Marketplace
- AI Price Recommendation
- Trust Score
- Wallet Integration
- Cardano Escrow
- Trust Ledger
- QR Verification
- Realtime Negotiation
- Multi-user Support
